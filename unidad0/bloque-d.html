Unidad 0 · Bloque D — Potenciación y Radicación

Repasamos el significado de las potencias y sus propiedades, y cómo las raíces se expresan como potencias fraccionarias. Este bloque conecta directamente con la Unidad 4 — Potenciación y Radicación.

📘 Recordatorio

  • Potencia: \(a^n\) significa multiplicar \(a\) por sí mismo \(n\) veces.
  • Exponente 0: \(a^0=1\) (\(a\neq0\)).
  • Exponente negativo: \(a^{-n}=\tfrac{1}{a^n}\).
  • Exponente fraccionario: \(a^{1/n}=\sqrt[n]{a}\).
  • Raíz cuadrada: \(\sqrt{a}\) es el número que al cuadrado da \(a\).

1) Calculá las potencias

  1. \(2^3\)
  2. \((-3)^2\)
  3. \((-3)^3\)
  4. \(5^0\)
  5. \(10^{-2}\)
Mostrar soluciones
  • \(2^3=8\)
  • \((-3)^2=9\)
  • \((-3)^3=-27\)
  • \(5^0=1\)
  • \(10^{-2}=\tfrac{1}{100}=0{,}01\)

2) Aplicá las propiedades de las potencias

Usá las reglas básicas: \(a^m·a^n=a^{m+n}\), \((a^m)^n=a^{m·n}\), \((a·b)^n=a^n·b^n\), \(a^m/a^n=a^{m-n}\).

  1. \(2^3·2^4\)
  2. \((3^2)^3\)
  3. \((2·5)^2\)
  4. \(\tfrac{5^6}{5^2}\)
  5. \(a^4·a^{-2}\)
Mostrar soluciones
  • \(2^3·2^4=2^7=128\).
  • \((3^2)^3=3^6=729\).
  • \((2·5)^2=2^2·5^2=4·25=100\).
  • \(5^6/5^2=5^{6-2}=5^4=625\).
  • \(a^4·a^{-2}=a^{4-2}=a^2\).

3) Calculá las raíces

  1. \(\sqrt{81}\)
  2. \(\sqrt{\tfrac{1}{4}}\)
  3. \(\sqrt{-9}\)
  4. \(\sqrt[3]{8}\)
  5. \(\sqrt[3]{-27}\)
Mostrar soluciones
  • \(\sqrt{81}=9\).
  • \(\sqrt{\tfrac{1}{4}}=\tfrac{1}{2}\).
  • \(\sqrt{-9}\) no es real.
  • \(\sqrt[3]{8}=2\).
  • \(\sqrt[3]{-27}=-3\).

4) Expresá como potencia con exponente fraccionario

  1. \(\sqrt{5}\)
  2. \(\sqrt[3]{7}\)
  3. \(\sqrt[4]{x^3}\)
  4. \(\sqrt[3]{a^6}\)
Mostrar soluciones
  • \(\sqrt{5}=5^{1/2}\).
  • \(\sqrt[3]{7}=7^{1/3}\).
  • \(\sqrt[4]{x^3}=x^{3/4}\).
  • \(\sqrt[3]{a^6}=a^{6/3}=a^2\).

5) Aplicación mixta

Resolvé usando propiedades de potencias y raíces:

  1. \(\sqrt{25·9}\)
  2. \(\sqrt{\tfrac{4}{9}}\)
  3. \(\sqrt[3]{8·27}\)
  4. \((\sqrt{16})^3\)
  5. \(\sqrt[4]{81}\)
Mostrar soluciones
  • \(\sqrt{25·9}=\sqrt{25}\sqrt{9}=5·3=15\).
  • \(\sqrt{\tfrac{4}{9}}=\tfrac{2}{3}\).
  • \(\sqrt[3]{8·27}=\sqrt[3]{8}\sqrt[3]{27}=2·3=6\).
  • \((\sqrt{16})^3=4^3=64\).
  • \(\sqrt[4]{81}=(81)^{1/4}=(3^4)^{1/4}=3\).

💡 Cuidado con esto

No confundas:
\(\sqrt{a+b}\neq\sqrt{a}+\sqrt{b}\). Ejemplo: \(\sqrt{9+16}=\sqrt{25}=5\neq3+4=7.\)
