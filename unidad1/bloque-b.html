Unidad 1 · Bloque B — Propiedades de las operaciones

Conmutativa, Asociativa y Distributiva: tres reglas que simplifican cuentas, álgebra y demostraciones.

Objetivos del Bloque B

  • Reconocer cuándo aplican las propiedades conmutativa, asociativa y distributiva.
  • Transformar y simplificar expresiones de forma segura.
  • Detectar errores típicos (qué no se puede hacer).

Conmutativa

El orden de los términos no cambia el resultado (solo en suma y producto).

Para todo a,b ∈ R:
a + b = b + a
a · b = b · a
No es conmutativa la resta ni la división.

Ejemplos

7 + 12 = 12 + 7 = 19

3·5 = 5·3 = 15

Error común

a − b ≠ b − a (en general). Ej.: 5−2=3 pero 2−5=−3.

Asociativa

La forma de agrupar (paréntesis) no altera el resultado (solo en suma y producto).

Para todo a,b,c ∈ R:
(a + b) + c = a + (b + c)
(a · b) · c = a · (b · c)
La resta y la división no son asociativas.

Ejemplos

(2 + 4) + 7 = 2 + (4 + 7) = 13

(3·7)·5 = 3·(7·5) = 105

Error común

(10 − 3) − 2 ≠ 10 − (3 − 2). Izq.=5, Der.=9.

Distributiva

Multiplicar por una suma (o resta) equivale a multiplicar por cada término y sumar (o restar).

Para todo a,b,c ∈ R:
a(b + c) = ab + ac
a(b − c) = ab − ac
(a + b)c = ac + bc

Ejemplo 1

2(x + 3) = 2x + 6

Se reparte el 2 a ambos términos.

Ejemplo 2

(a + b)(x + y) = ax + ay + bx + by

Producto de binomios = “todos con todos”.

Ejemplo 3

−2(x + y − 5) = −2x − 2y + 10

Atención al signo negativo: cambia todos los signos dentro.
Errores típicos a evitar
  • a(b + c) ≠ ab + c (falta distribuir al segundo término).
  • (a + b)² ≠ a² + b² (falta el término 2ab). Correcto: a² + 2ab + b².
  • −(x − y) ≠ −x − y (correcto: −x + y).

Estrategias rápidas

  • Reordenar sumas o productos (conmutativa) para juntar “parejas convenientes”.
  • Reagrupar (asociativa) para calcular primero lo más simple.
  • Expandir o factorizar (distributiva) según convenga: a(b+c) ↔ ab+ac.
  • Control de signos: un “−” delante de paréntesis cambia todos los signos.

Mini-actividad 2 · Simplifica correctamente

  1. 2(x + 5) − (x − 3)
  2. (a + b)(a − b)
  3. −(3y − 2) + 4(2 − y)
  4. (2m + 1)²
  5. 3x − 2x + 5 − (x − 4)
Mostrar soluciones paso a paso
  1. 2x + 10 − x + 3 = x + 13
  2. a² − b² (producto notable) = (a − b)(a + b) o expandido a² − b²
  3. −3y + 2 + 8 − 4y = −7y + 10
  4. (2m + 1)(2m + 1) = 4m² + 4m + 1 → 4m² + 4m + 1
  5. 3x − 2x + 5 − x + 4 = (3x − 2x − x) + 9 = 9
