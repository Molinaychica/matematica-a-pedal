Unidad 2 · Capítulo 2 — Operaciones con expresiones algebraicas

Sumar, restar y multiplicar polinomios aplicando las propiedades básicas.

Objetivos

  • Identificar términos semejantes y combinarlos correctamente.
  • Aplicar la distributiva para multiplicar expresiones.
  • Reconocer errores comunes en los signos y paréntesis.

1️⃣ Suma y resta

Solo se pueden sumar o restar términos semejantes, es decir, los que tienen las mismas variables y exponentes.

Ejemplo: (3x² + 5x − 2) + (4x² − 3x + 7) = 7x² + 2x + 5.

Si hay un signo “−” delante de paréntesis, cambia los signos de los términos internos antes de combinar.

Ejemplo: (6x − 3y) − (2x + 4y) = 6x − 3y − 2x − 4y = 4x − 7y.

2️⃣ Multiplicación de polinomios

Aplicamos la propiedad distributiva: cada término del primer polinomio multiplica a cada término del segundo.

Ejemplo 1: (a + b)(c + d) = ac + ad + bc + bd. Ejemplo 2: (x + 2)(x + 5) = x² + 7x + 10.

3️⃣ Multiplicación con más de dos términos

Primero se multiplican dos factores, y luego el resultado por el siguiente (usando asociativa).

Ejemplo: (x + 1)(x + 2)(x + 3) = (x² + 3x + 2)(x + 3) = x³ + 6x² + 11x + 6.

Mini-actividad · Suma, resta y producto

  1. (4x² − 3x + 7) + (−2x² + 5x − 1)
  2. (a + b)(a − b)
  3. (2x + 3)(x − 5)
  4. (x + 1)(x² + x + 1)
Mostrar soluciones
  1. (4x² − 3x + 7) + (−2x² + 5x − 1) = 2x² + 2x + 6.
  2. (a + b)(a − b) = a² − b² (diferencia de cuadrados).
  3. (2x + 3)(x − 5) = 2x² − 10x + 3x − 15 = 2x² − 7x − 15.
  4. (x + 1)(x² + x + 1) = x³ + x² + x + x² + x + 1 = x³ + 2x² + 2x + 1.
