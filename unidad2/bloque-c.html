Unidad 2 · Bloque C — Productos notables

Reglas de multiplicación especiales que permiten desarrollar o factorizar sin hacer todos los pasos.

Objetivos

  • Recordar y aplicar los principales productos notables.
  • Reconocer cuándo una expresión corresponde a un caso notable.
  • Usar estos productos como herramienta rápida para el desarrollo y la factorización.

1️⃣ Cuadrado de una suma

(a + b)2 = a2 + 2ab + b2

Ejemplo: (x + 5)2 = x2 + 10x + 25.

2️⃣ Cuadrado de una diferencia

(a − b)2 = a2 − 2ab + b2

Ejemplo: (x − 3)2 = x2 − 6x + 9.

3️⃣ Diferencia de cuadrados

(a + b)(a − b) = a2 − b2

Ejemplo: (x + 2)(x − 2) = x2 − 4.

4️⃣ Cubos perfectos

(a + b)3 = a3 + 3a2b + 3ab2 + b3

(a − b)3 = a3 − 3a2b + 3ab2 − b3

Ejemplo: (x − 1)3 = x3 − 3x2 + 3x − 1.

5️⃣ Suma y diferencia de cubos

a3 + b3 = (a + b)(a2 − ab + b2)

a3 − b3 = (a − b)(a2 + ab + b2)

Mini-actividad · Reconocé el producto notable

  1. (2x + 3)2
  2. (x − 5)(x + 5)
  3. (a + 2b)3
  4. x3 − 8
Mostrar soluciones
  1. (2x + 3)2 = 4x2 + 12x + 9 (cuadrado de una suma)
  2. (x − 5)(x + 5) = x2 − 25 (diferencia de cuadrados)
  3. (a + 2b)3 = a3 + 6a2b + 12ab2 + 8b3 (cubo de una suma)
  4. x3 − 8 = (x − 2)(x2 + 2x + 4) (diferencia de cubos)
