Unidad 2 · Capítulo 5 — Fracciones algebraicas

Cuando el numerador y/o denominador son expresiones algebraicas, las reglas de fracciones siguen valiendo con atención a los ceros del denominador.

Objetivos

  • Comprender el concepto de fracción algebraica y su dominio.
  • Aprender a simplificar dividiendo por factores comunes.
  • Aplicar operaciones (× ÷ + −) entre fracciones algebraicas.

1️⃣ Definición y dominio

Una fracción algebraica es una expresión de la forma A(x)/B(x), donde A y B son polinomios y B(x) ≠ 0.

Ejemplo 1: (3x + 6)/(x² − 4)  Ejemplo 2: (2a² − 8a)/(a² + a − 6)

El dominio excluye los valores que anulan B(x).

2️⃣ Simplificación

Se factorizan numerador y denominador y se eliminan factores comunes (no términos, solo factores).

Ejemplo: (3x + 6)/(x² − 4) = 3(x + 2)/[(x + 2)(x − 2)] = 3/(x − 2), con x ≠ ±2.

3️⃣ Multiplicación y división

  • Multiplicación: (a/b) × (c/d) = (ac)/(bd)
  • División: (a/b) ÷ (c/d) = (a/b) × (d/c)
Ejemplo: (2x)/(3y) × (9y²)/(4x²) = (18y)/(12x) = 3y/(2x)

4️⃣ Suma y resta

Requieren denominador común (igual que las fracciones numéricas).

Ejemplo: 1/x + 2/x² = (x + 2)/x²

Si los denominadores son distintos, usamos el mínimo común múltiplo (MCM).

Ejemplo: (3/(x + 1)) − (2/x) → MCM = x(x + 1) → [3x − 2(x + 1)]/[x(x + 1)] = (x − 2)/[x(x + 1)]

5️⃣ Restricciones del dominio

Recordá indicar qué valores anulan el denominador y por tanto no pertenecen al dominio de la fracción.

Ejemplo: (2x − 1)/(x² − 9) → x ≠ 3, x ≠ −3.

Mini-actividad · Simplificación y operaciones

  1. (4x² − 12x)/(8x)
  2. (x² − 9)/(x − 3)
  3. (a² − b²)/(a − b)
  4. (3x)/(x + 2) + (2)/(x + 2)
  5. (x² − 4)/(x² − x − 6)
Mostrar soluciones
  1. (4x² − 12x)/(8x) = 4x(x − 3)/(8x) = (x − 3)/2
  2. (x² − 9)/(x − 3) = (x − 3)(x + 3)/(x − 3) = x + 3, x ≠ 3
  3. (a² − b²)/(a − b) = (a + b), a ≠ b
  4. Denominador común (x + 2): (3x + 2)/(x + 2)
  5. (x² − 4)/(x² − x − 6) = (x − 2)(x + 2)/[(x − 3)(x + 2)] = (x − 2)/(x − 3), x ≠ −2, 3

Resumen de la Unidad 2

  • El lenguaje algebraico permite representar problemas generales con letras.
  • La factorización es la herramienta clave para simplificar y resolver expresiones.
  • Las fracciones algebraicas siguen las mismas reglas que las numéricas, con atención al dominio.
