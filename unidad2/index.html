Mini-evaluación (10 ítems)

Resolvé y luego desplegá “Mostrar soluciones” para autocorregirte.

  1. Clasificá y hallá el grado: 3x²y³ − 4xy.
  2. Simplificá: (5x² − 3x + 2) + (2x² − x − 4).
  3. Expandí: (x + 3)².
  4. Factorizá: 9x² − 16.
  5. Reconocé el caso notable en a³ + 27.
  6. Simplificá: (6x³ − 9x²)/(3x²).
  7. Calculá: (x + 2)(x² − 2x + 4).
  8. Simplificá: (x² − 9)/(x − 3).
  9. Sumá: 3/(x + 1) + 2/x.
  10. Indicá el dominio de (2x − 1)/(x² − 4).
Mostrar soluciones
  1. Polinomio de dos términos, grado 5 (2 + 3 = 5).
  2. 7x² − 4x − 2.
  3. x² + 6x + 9.
  4. (3x + 4)(3x − 4).
  5. (a + 3)(a² − 3a + 9).
  6. (6x³ − 9x²)/(3x²) = 2x − 3.
  7. x³ + 2x² − 2x + 4x + 2x² − 4x + 8 = x³ + 8.
  8. (x² − 9)/(x − 3) = x + 3 ( x ≠ 3 ).
  9. MCM = x(x + 1) → (3x + 2(x + 1))/[x(x + 1)] = (5x + 2)/[x(x + 1)].
  10. x ≠ 2, −2.
