Mini-evaluación (10 ítems)

Resolvé y luego desplegá “Mostrar soluciones” para autocorregirte.

  1. Clasificá y hallá el grado: \(3x^2y^3 - 4xy\).
  2. Simplificá: \((5x^2 - 3x + 2) + (2x^2 - x - 4)\).
  3. Expandí: \((x + 3)^2\).
  4. Factorizá: \(9x^2 - 16\).
  5. Reconocé el caso notable en \(a^3 + 27\).
  6. Simplificá: \((6x^3 - 9x^2)/(3x^2)\).
  7. Calculá: \((x + 2)(x^2 - 2x + 4)\).
  8. Simplificá: \((x^2 - 9)/(x - 3)\).
  9. Sumá: \( \frac{3}{x + 1} + \frac{2}{x} \).
  10. Indicá el dominio de \( \frac{2x - 1}{x^2 - 4} \).
Mostrar soluciones
  1. Polinomio de dos términos, grado \(2+3=5\).
  2. \(7x^2 - 4x - 2\).
  3. \(x^2 + 6x + 9\).
  4. \((3x + 4)(3x - 4)\).
  5. \((a + 3)(a^2 - 3a + 9)\).
  6. \((6x^3 - 9x^2)/(3x^2) = 2x - 3\).
  7. \((x + 2)(x^2 - 2x + 4) = x^3 + 8\).
  8. \((x^2 - 9)/(x - 3) = x + 3\) (con \(x \ne 3\)).
  9. MCM \(= x(x+1)\) → \(\dfrac{3x + 2(x + 1)}{x(x+1)} = \dfrac{5x + 2}{x(x+1)}\).
  10. Dominio: \(x \ne 2, -2\).
