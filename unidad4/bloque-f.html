Poné en práctica potenciación, radicación y notación científica con problemas de dificultad creciente. Al final, verificá tus respuestas y reflexioná sobre en qué propiedad o idea te apoyaste en cada caso.

A) Cálculo y simplificación

  1. Simplificá: 73 · 7−5.
  2. Escribí como una sola potencia: (24 · 2−9) / 2−3.
  3. Evaluá y dejá en forma exacta: 813/4.
  4. Simplificá: √(98) − 2√(8) + 3√(18).
  5. Reescribí como potencia: 3√(x8).
Mostrar soluciones
  1. 73−5 = 7−2 = 1/49.
  2. 24−9−(−3) = 2−2 = 1/4.
  3. 81 = 34 ⇒ 813/4 = (34)3/4 = 33 = 27.
  4. √98=7√2, √8=2√2, √18=3√2 ⇒ 7√2 − 2·2√2 + 3·3√2 = 7√2 − 4√2 + 9√2 = 12√2.
  5. 3√(x8) = x8/3 = x2 + 2/3 = x2·x2/3.

B) Notación científica

  1. Convertí a notación científica: 0,000 000 092.
  2. Expresá 3,6·107 + 1,4·107 en notación científica.
  3. Calculá: (2,5·10−4) · (4·106).
  4. Dividí: (7,2·108) / (1,8·103).
Mostrar soluciones
  1. 9,2·10−8.
  2. (3,6+1,4)·107 = 5,0·107.
  3. 2,5·4 · 10−4+6 = 10,0·102 = 1,0·103.
  4. (7,2/1,8)·108−3 = 4·105.

C) Aplicadas

  1. Biología: Un cultivo crece de 2·106 a 6·107 bacterias. ¿Cuántas veces se multiplicó la población?
  2. Astronomía: La distancia es 4,2·1016 m. Expresá en km (notación científica).
  3. Química: Una concentración es 3,0·10−6 mol/L. Multiplicala por un factor 4,0·102. ¿Resultado?
  4. Ingeniería: Un sensor lee 2,4·10−3 A y luego 6,0·10−4 A. ¿Qué relación hay entre ambas lecturas?
Mostrar soluciones
  1. (6·107)/(2·106) = 3·10 = 30 veces.
  2. 4,2·1016 m = 4,2·1013 km.
  3. (3,0·10−6)·(4,0·102) = 12,0·10−4 = 1,2·10−3 mol/L.
  4. (2,4·10−3)/(6,0·10−4) = 4 → la primera es 4 veces mayor.

Autoevaluación conceptual (verdadero/falso + explica)

  1. (ab)n = an + bn.
  2. Si n es par, √[n](−8) ∈ ℝ.
  3. a−2 = 1/a2, con a ≠ 0.
  4. √(a·b) = √a · √b para a,b ≥ 0.
  5. 00 lo definimos como 1 en este curso.
Mostrar soluciones
  1. Falso. (ab)n = anbn.
  2. Falso. Con índice par no hay raíz real de negativo.
  3. Verdadero.
  4. Verdadero (en ℝ y con a,b ≥ 0).
  5. Falso. 00 se considera indeterminado en esta unidad.
